# bash_tutorial
Shell 这个单词的原意是“外壳”，跟 kernel（内核）相对应，比喻内核外面的一层，即用户跟内核交互的对话界面。  
Shell有很多种，这里指Linux Bash。

## 参考文档
[bash脚本教程](https://wangdoc.com/bash/intro)
